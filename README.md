**# supermarket-billing-system**

A lightweight and easy-to-use Java application for generating supermarket bills. Built with clean code, simple design, and practical functionality.

**What It Does..?**

Manages products (add, view, update, delete)

Generates customer bills

Calculates totals, discounts, and tax

Provides clear and accurate invoices

Stores product/billing data
